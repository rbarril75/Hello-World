Hello-World
===========

A collection of Hello World programs.

##Usage
* Java
  * javac HelloWorld.java
  * java HelloWorld
* C
  * gcc -o HelloWorld HelloWorld.c
  * ./HelloWorld
* C++
  * g++ -o HelloWorld HelloWorld.cpp
  * ./HelloWorld
* Python
  * python HelloWorld.py
* Ruby
  * ruby HelloWorld.rb
* Perl
  * perl HelloWorld.pl
* PHP
  * php HelloWorld.php
* JavaScript
  * node HelloWorld.js
* Bash
  * bash HelloWorld.sh
